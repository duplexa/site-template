# Site Template

A fast, SEO-ready static site template built with Astro, deployed to Cloudflare Pages.

## Setup Checklist

When creating a new site from this template, update these files:

1. **`astro.config.mjs`** — Set `site` to your domain (e.g. `https://yourdomain.com`)
2. **`public/robots.txt`** — Update the `Sitemap:` URL to match your domain
3. **`src/site.config.ts`** — Set your site name, description, and footer text
4. **Google AdSense** — If using ads, add your AdSense script to `src/components/BaseHead.astro`
5. **Google Search Console** — After deploying, submit your sitemap at `https://yourdomain.com/sitemap-index.xml`

## SEO Notes

- `trailingSlash: 'never'` is set in the Astro config to prevent duplicate URL indexing (with and without trailing slash)
- Canonical URLs are generated automatically and normalized to strip trailing slashes
- The sitemap is auto-generated by `@astrojs/sitemap` at build time
- Set up a Cloudflare redirect rule to 301 redirect trailing-slash URLs to non-trailing-slash versions

## Commands

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | Install dependencies                        |
| `npm run dev`     | Start local dev server at `localhost:4321`   |
| `npm run build`   | Build production site to `./dist/`           |
| `npm run preview` | Preview build locally before deploying       |
